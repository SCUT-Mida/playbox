# 凡人修仙录 · Mortal Cultivation

一款致敬《凡人修仙传》的**文字修仙挂机**网页游戏。从一介凡人起步，修炼、突破境界、探索机缘、炼丹炼器、渡劫飞升，体验一段完整的修仙旅程。

技术栈：**纯原生 HTML + CSS + JavaScript（无框架）**，移动端竖屏单列设计，Canvas 都不用——一切皆 DOM 卡片。体积小（构建后 JS ≈ 60KB），加载快。

## 本地运行

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5174
npm run build    # 生产构建到 dist/
npm run test     # 纯逻辑自测（300+ 断言，不依赖浏览器）
```

也可由主框架（落地页）以 `createGame(parent)` 动态挂载，无需独立部署。

## 核心玩法

- **创角**：进入游戏先选男女，随机生成**灵根（等级·组合·属性）/ 年龄 / 天赋（1~3 个）/ 气运 / 出身**与**形象**（10+ 预设头像），不满意可反复重随，称心后取名入道。天赋影响修炼、突破、炼丹、攻防与每月行动力；气运影响奇遇保底与稀有概率。
- **人物档案**：点状态栏头像随时查看形象、灵根（等级·组合·属性）、气运、年龄寿元、天赋、出身与各项属性，及灵根与功法/法宝的**契合度**。
- **境界体系**：凡人 → 炼气（13 层）→ 筑基 → 结丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → **飞升**，完整还原原作层级；**每境皆有寿元上限，境界越高寿元越长**（飞升与天同寿）。
- **境界总纲**：状态栏境界徽章只呈现大境界，点击展开完整的「境界轴」说明（凡人→飞升全层级、渡劫方式、修炼系数、寿元），可上下滑动，当前境界有明显标注。
- **灵根三轴**：开局随机灵根由三个维度共同决定——**等级**（伪 / 人 / 地 / 天 / 异灵根）、**组合**（单 / 双 / 三 / 四 / 五灵根，越少越精纯）、**属性**（金木水火土五行，或风雷冰等异属性）。不同属性灵根与**功法 / 法宝有契合度**：属性一致时修炼速度、攻防与突破成功率额外加成（洗髓丹可提升等级档次）。
- **修炼**：被动实时挂机攒修为；主动修炼消耗灵力与活力、有概率「顿悟」三倍暴击。灵根等级与组合共同决定修炼速度与突破成功率。
- **突破**：小境界掷骰判定（可服突破丹）；大境界需**渡心魔**（神识多轮）或**渡天劫**（多轮雷火，可硬抗 / 祭法宝 / 服丹）。
- **探索**：随机场景（灵山 / 洞府 / 遗迹 / 野外）+ 加权事件池——采药挖矿、遭遇妖兽、修士争斗、仙人遗府、空间裂缝、心魔幻境……含**保底机制**，连黑必有奇遇。
- **战斗**：文字回合制，攻击 / 防御 / 法宝技能 / 丹药 / 逃跑，伤害含五行克制与随机浮动。
- **坊市**：货架随机刷新、价格浮动；魔劫入侵时物价飙升。可买丹药、材料、法宝、功法、配方，半价回收。每件商品都有**一句话介绍**（丹药效果量 / 法宝属性 / 功法加成 / 配方产出），点开查看详情再决定购买。
- **炼丹 / 炼器**：按丹方器图投入材料，神识与难度决定成率，产出有**品质波动**（低阶 / 正常 / 极品双份）。
- **宗门**：拜入青云宗 / 万剑宗 / 丹霞谷等 **11 大宗门**之一，接取**每月宗门任务**（每月 5 个：修炼 / 探索 / 降妖 / 炼制 / 突破），完成提升**宗门声望**；另可自主接取**悬赏挑战**（难度与奖励随境界提升，灵石 + 声望 + 高境界物品）。凭「境界 + 声望」双门槛晋升**宗门称号**（记名弟子 → 外门弟子 → 内门弟子 → 真传弟子 → 护法长老 → 太上长老 → 一代宗主），称号提供修炼加成，并按称号领取**每月宗门俸禄**。
- **道友（NPC 好感度）**：云游寻访结识各路修士，赠礼（喜好物增益更高）提升好感，经历点头之交 → 投缘之交 → 莫逆之交 → 生死道侣数阶。好感达「投缘之交」即可**结伴探险**——奖励随境界与道友大幅放大，远胜独行；莫逆之交及以上的道友还提供修炼加成（每月每道友限同游一次）。
- **音效**：Web Audio 现场合成的短音（无音频文件），覆盖修炼 / 顿悟 / 突破 / 战斗 / 探索 / 领奖 / 结识等事件；状态栏一键 🔊/🔇 开关，记忆偏好。
- **活力补给**：坊市常售**培元丹 / 百草灵酒 / 玉髓仙茶**等回复活力的丹药，缓解每月行动力紧张。
- **背包**：分类物品、法宝装备与耐久、储物袋扩容；功法 / 称号提供被动加成。
- **成就与称号**：空血反杀、十连突破、炼丹宗师、白日飞升等隐藏成就。
- **混沌事件**：极低概率触发世界级「灵潮爆发」（修炼翻倍）、「魔劫入侵」（物价紊乱）。
- **年龄与轮回**：岁月以**自然月为周期**流转——每月行动力回满、宗门任务刷新，**年龄随之增长**；状态栏实时显示年龄，逾越当前境界寿元即**大限将至**（可突破延寿）。大限时可**轮回重修一次**：境界修为归零、年龄重置，但携带前世灵根档次与属性、气运、一项天赋，更获「前世记忆」修炼加成；此后年龄标注**（轮回）**。
- **每月行动力**：主动修炼 / 探索 / 炼制消耗**活力**（行动力），跨自然月自动回满；活力耗尽时仅能挂机被动修炼、逛坊市等「部分事情」，或「闭关静修」即刻恢复一次（每月限一次），符合修仙挂机节奏。
- **多存档**：支持 **5 个存档槽**，可新建 / 进入 / 删除 / 重开；旧版单存档首启自动迁移到槽 1。仍保留导出/导入字符串与**离线修炼收益**（上限 8 小时）。
- **自动挂机**：在「设置」中开启后，主角按你勾选的**倾向**（修炼 / 探索 / 突破 / 炼制 / 结伴 / 寻访，可多选）与各自**权重**，每「周期」自动修行——遭遇、战斗、渡劫全部自动处置，无需手动操作；打开任意弹窗时自动暂停，不与你的操作冲突。被动修炼始终在累积，真正实现「挂机修仙」。

```
src/
  config.js          境界/灵根/数值公式（纯函数）
  data/
    items.js         丹药/材料/法宝/功法/杂物图鉴

    recipes.js       丹方与器图

    sect.js          宗门 / 宗门称号（境界+声望门槛）/ 每日宗门任务池 / 悬赏挑战模板

  core/
    rng.js           可注入随机源（种子化/测试）
    player.js        玩家状态、衍生属性、背包、装备、创角、活力
    cultivate.js     修炼（被动/主动/顿悟）
    explore.js       探索抽取 + 奖励结算 + 混沌 buff

    breakthrough.js  突破 + 心魔/天劫考验

    market.js        坊市（含商品介绍/详情）
    achievements.js  成就与称号
    sect.js          宗门：入宗/每日任务/声望/称号/每日俸禄/悬赏挑战
    npc.js           道友：寻访/赠礼/好感/组队探险
    autoplay.js      自动挂机：按倾向与权重每周期自动修行（自动处置遭遇/战斗/渡劫）
    sfx.js           音效：Web Audio 合成 + 开关（无音频文件）
    save.js          多槽存档/导入导出/离线结算
  ui/
    dom.js           轻量 h() DOM 辅助
    style.css        竖屏移动端样式
    app.js           UI 控制器（存档选择/创角/状态栏/五页/弹窗/循环）

scripts/smoke-dom.mjs    jsdom 冒烟测试（需 npm i jsdom，覆盖存档/创角/活力 UI 流程）
```

## 设计原则

1. **碎片时间友好**：单次操作数秒，修炼可挂机。
2. **正反馈强烈**：突破、爆宝、奇遇有醒目动效与文案。
3. **风险与机遇并存**：探索可能遇险，也可能一夜暴富。
4. **足够随机但不致死**：保底机制、复活丹、重修 buff 给予挽回途径。
5. **修仙味要正**：境界名、事件文案模仿原作语境。
